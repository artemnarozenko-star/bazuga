# 🚗 Автосалон

Клиент-серверное приложение для управления и продажи автомобилей на веб-сайте автосалона.

## 📖 О проекте

Данная система позволяет менеджерам вести единую базу данных со всеми данными системы. 
Каждый пользователь может авторизоваться на сайте, чтобы отслеживать ранее добавленные автомобили в корзину или в список «понравившихся».

В приложении реализована удобная фильтрация для быстрого поиска автомобилей, подходящих под нужды каждого клиента. 
Также присутствуют популярные категории для навигации.

---

## 🛠 Технологии

*   **Frontend:** HTML, CSS
*   **База данных:** MySQL
